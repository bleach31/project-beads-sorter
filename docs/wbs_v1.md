# WBS v1 — ビーズ仕分け機

- [ ] **1.1 全体アーキテクチャ設計** — ソフト・ハード・CAD間の接続・通信方式を決定する
- [ ] **1.2 フィーダー機構設計** — ロート+スクリュー+テフロンチューブの形状・寸法を検討する（SPEC_002）
- [ ] **1.3 認識部設計** — Pi Cameraの取付位置・照明条件を決定する（SPEC_001）
- [ ] **1.4 排出・ターンテーブル機構設計** — サーボ排出+回転テーブルの配置・寸法を検討する（SPEC_003, SPEC_004）
- [ ] **1.5 回路設計** — モータードライバ・電源配線・ラズパイGPIO割当を決定する

- [ ] **2.1 電子部品購入** — ステッピングモーター、サーボ、DCモーター、Pi Camera、ケーブル類

## 5. ソフトウェア開発

- [ ] **5.1 カメラ画像取得・色認識** — Pi Cameraで撮影→色判定ロジック実装（vision モジュール）
