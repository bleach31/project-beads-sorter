システム要件
============

.. req:: 自動仕分け
   :id: REQ_001
   :status: open
   :tags: system

   ビーズを色ごとに自動仕分けできること。

.. req:: 仕分け色数
   :id: REQ_002
   :status: open
   :tags: system

   10色を識別・仕分けできること。

.. req:: 対象ビーズ
   :id: REQ_003
   :status: open
   :tags: system

   アイロンビーズ（φ5mm×h5mm）およびアクアビーズ まるビーズに対応すること。

.. req:: ビーズ供給
   :id: REQ_004
   :status: open
   :tags: feeder

   ビーズを1個ずつ供給できること。

.. req:: 3Dプリント筐体
   :id: REQ_005
   :status: open
   :tags: mechanical

   筐体は3Dプリントで製作可能であること。

.. req:: Type-C給電
   :id: REQ_006
   :status: open
   :tags: electrical

   電源はType-C給電を前提とすること。

.. req:: 段階的改善
   :id: REQ_007
   :status: open
   :tags: system

   3台製作し段階的に改善すること（v1→v2→v3）。
